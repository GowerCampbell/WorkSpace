## 3. How to Contribute ğŸ› ï¸
### **1. Fork and Clone the Repository**
1. Navigate to the project repository on GitHub.
2. Click the **Fork** button (top right) to create a copy under your account.
3. Clone your forked repository:
   ```sh
   git clone https://github.com/your-username/repository-name.git
   ```
4. Change to the project directory:
   ```sh
   cd repository-name
   ```

### **2. Create a New Branch**
```sh
git checkout -b feature-branch-name
```

### **3. Make Your Changes and Commit**
- Modify the necessary files.
- Test your changes (if applicable).
- Add your changes to staging:
  ```sh
  git add .
  ```
- Commit your changes with a meaningful message:
  ```sh
  git commit -m "Fixed issue #123: Improved UI for login page"
  ```

### **4. Push Your Changes and Create a Pull Request (PR)**
```sh
git push origin feature-branch-name
```
- Go to the original repository on GitHub.
- Click **New Pull Request**.
- Provide a clear description of the changes.
- Submit for review!
